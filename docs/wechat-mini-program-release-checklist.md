# 露米绘画AI微信小程序上线清单

## 代码包

- 使用微信开发者工具导入 `apps/mobile/dist/build/mp-weixin`。
- 上传前运行 `pnpm verify:mini-program-release`。
- 上传版本号使用 `1.0.0`，项目备注写清本次为首次正式提审。
- 不要关闭微信开发者工具中的合法域名校验。

## 微信公众平台必须完成

1. 完成小程序备案，并确认备案主体与小程序主体一致。
2. 核对名称、头像、简介和服务类目，类目需覆盖 AI 图片生成、图片工具和用户作品展示。
3. 在“服务内容声明/用户隐私保护指引”中如实声明：
   - 微信登录产生的账号标识；
   - 用户主动选择的头像、昵称、相册图片或相机图片；
   - 提示词、参考图、生成图、作品标题和描述；
   - 反馈内容、反馈图片及用户主动填写的联系方式；
   - 订单、支付状态、积分和会员权益记录；
   - 使用微信、阿里云 OSS/CDN、AI 图片生成服务商完成服务所必需的数据处理。
4. 发布《小程序用户隐私保护指引》，并确认小程序“设置”中的官方指引入口可以打开。
5. 配置服务器域名：
   - request 合法域名：`https://ejoyflie.cloud`
   - request/upload 合法域名：`https://lumidraw.oss-cn-beijing.aliyuncs.com`
   - downloadFile 合法域名：`https://img.ejoyflie.cloud`
6. 配置并测试客服人员，确保“设置—联系客服”能进入真实客服会话。
7. 添加至少一个体验成员和一个审核专用测试账号；在提审备注中写明登录无需手机号，并给出完整体验路径。

## 提审前必须处理的业务合规项

- 积分充值、会员和 AI 生图额度属于虚拟权益。提交审核前，需要在微信公众平台完成“小程序虚拟支付”开通并改用官方虚拟支付接口；未完成前应临时隐藏充值、会员价格、购买按钮和促销走马灯，不能继续使用普通小程序支付作为正式上线方案。
- “邀请好友获得积分”存在诱导分享审核风险。提审版本应关闭邀请奖励入口、分享文案和相关走马灯；后续只有在确认活动符合平台规范后再开放。
- AI 生成内容和用户发布内容应保持人工审核开启。未接入真实的微信文字/图片内容安全能力前，不要开启自动发布。
- 公开页面不得保留过期活动、测试文案、测试账号、无效按钮或无法兑现的会员权益。

## 提审备注建议

> 本小程序提供 AI 图片创作、作品管理与经审核后的作品展示。登录使用微信静默登录，不获取手机号；头像、昵称、相册和相机仅在用户主动操作时使用。AI 生成内容均有明确标识，公开作品进入人工审核流程。设置页提供隐私保护指引、客服、反馈和账号注销入口。

## 上线后检查

- 审核通过后先发布，再用非开发者微信号进行一次完整回归。
- 检查登录、创作、图生图、任务后台运行、作品发布与审核、保存图片、客服、反馈和账号注销。
- 检查生产环境 mock 登录、模拟生成、模拟支付均关闭。
- 检查支付回调、OSS/CDN、AI 平台余额、人工审核队列、退款与积分补偿日志。
- 在管理后台关闭过期公告，并每天检查失败任务、内容举报和用户反馈。
